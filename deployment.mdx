## Docker 部署

### 镜像地址

- `ghcr.io/eis4ty/rokid-rizon-agent-bridge:latest`

> 首次拉取需要登录 GitHub Container Registry（GHCR）。

### 构建镜像

```bash
docker build -t rokid-rizon-agent-bridge .
```

### 运行容器

```bash
docker run -p 3000:3000 \
  -v $(pwd)/bindings.json:/app/bindings.json \
  -v $(pwd)/settings.json:/app/settings.json \
  -v $(pwd)/logs.json:/app/logs.json \
  rokid-rizon-agent-bridge
```

> 建议挂载 `bindings.json` 与 `settings.json` 以持久化配置。

### 使用已发布镜像

```bash
docker login ghcr.io
docker pull ghcr.io/eis4ty/rokid-rizon-agent-bridge:latest
```

```bash
docker run -p 3000:3000 \
  -v $(pwd)/bindings.json:/app/bindings.json \
  -v $(pwd)/settings.json:/app/settings.json \
  -v $(pwd)/logs.json:/app/logs.json \
  ghcr.io/eis4ty/rokid-rizon-agent-bridge:latest
```

## Node 生产部署

```bash
npm install --production
npm run build
npm run start
```

确保 `bindings.json`、`settings.json`、`logs.json` 有可写权限。
